HTML 5

H1 – serve para colocar títulos 
p- mostra o texto comum um parágrafo na pagina
div- envolver diversos elementos do código assim para aplicar CSS (geralmente os browsers pulam uma linha para separar os elementos de uma div e os demais). As divs não apresentam nenhuma mudança visual na tela
span- semelhante a div, serve para envolver um ou mais elementos e não causa uma quebra de linha. Serve para fazer uma estilização mais centralizada
ul- serve para uma lista não ordenada por bolinhas
ol – lista ordenada por números
href = manda para um link
img src= serve para mostra uma imagem

<!DOCTYPE html> = serve para avisar o browser que é um documento html
<html tag = “pt-br”> - serve para informa o programador que tipo de linguagem está esse documendo

Head – são as outras tag que não vão ser mostradas ao usuário.
Body- tag que envolve todas as tags que serão mostradas no browser
Meta tag – são tags que ficam dentro do head e que não aparecem diretamente na página mas nos ajudam a dar informações sobre ela para o browser e os mecanismos de busca como o Google
	Meta charset=”UTF-8” – diz ao browser que vamos usar o padrão encode UTF-8.
	Meta name=”description” – informa qual o conteúdo que irá apresentar naquela pagina
	Meta name=”Author” -quem é o ator daquela pagina

Form – permite submeter informações para um servidor. Por exemplo, enviar os dados de um formulário para serem salvos em um banco de dados
Action- é para onde a gente vai enviar o formulário a URL
Method – diz qual o método vamos usar
		Get- geralmente usado para pegar informações
		Post – geralmente usado para gravar informações no servidor
<input type=”text” name=”q” required> -O required é basicamente o sistema não deixa você lançar o form se esse input não estiver preenchido 

Label- é um texto que geralmente acompanha o input.



Em um select existe um 
	<option desable selected> - um option que não vale nada e que pode ser usado como placeholder
